Status: `planned`
Owner: `user + OpenCode`

## Goal

Raise the MVP from feature-complete to stable for repeated personal use.

## Scope

- Performance tuning for lists and images
- Android TV focus bug fixing
- Better reconnect, timeout, and invalid-config handling
- Large-library behavior checks
- Quality pass on loading, error, and empty states
- Polish of Home and playback transitions

## Deliverables

- Beta-quality build candidate
- Known limitations list
- Prioritized bug and polish backlog

## Exit Criteria

- App feels stable enough for daily self-use
- Core flows survive bad network and empty-data cases
- TV navigation regressions are reduced to minor issues

## Notes
